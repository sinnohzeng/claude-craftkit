# 锚本 11：北京市 2024 年政府工作报告「对标」标准用法

**来源**：北京市人民政府，2024 年 1 月 29 日发布
**URL**：https://www.beijing.gov.cn/gongkai/jihua/zfgzbg/202401/t20240129_3547363.html
**类型**：省级政府工作报告
**特征**：「对标 + 国际先进 / 一流 / 国际标杆」三种标准用法

---

## 原文（节选 1，对标国际先进水平）

> 大力支持产业数字化。深入实施"新智造 100"工程，全面开展制造业企业数字化转型，加快培育标杆性"数字领航"企业。促进平台经济有序竞争、创新发展，推动先进数字技术向中小企业深度普及，构建开放共享、充满活力的创新生态。提升人工智能底层技术和基础底座自主可控能力，**推动人工智能模型对标国际先进水平**，加快在政务、医疗、教育、工业、生活服务等领域应用，保持人工智能研发应用领先水平。

## 原文（节选 2，对标世界银行评估框架）

> 打造营商环境"北京服务"品牌。落实全面优化营商环境意见，**对标世界银行新营商环境评估框架**，营造市场化、法治化、便利化、国际化一流营商环境。编制本市行政许可事项实施规范，深化"一业一证"和"一件事"集成服务改革，深入推进一体化综合监管，扩大非现场监管占比，完善全国市场监管数字化试验区建设。

## 学什么

1. **「对标 + 具体对象」是党政标准用法**：对标国际先进水平、对标世界银行框架、对标一流、对标先进——每次「对标」都接具体对象，不是空泛的"对标"
2. **「对标」与「对照」可互换**：在党政公文中两者都是合法用法。咨询报告若与甲方文件混用，统一为"对照"以与互联网"对标 KPI / 对标头部产品"区隔
3. **「对标 + 国际 + 一流 / 先进 / 标杆」组合高频**：政府工作报告每年多次出现，属于规范的政治表达
4. **不要把「对标」误判为大厂黑话**：xuan-jiang v4.3 §1.5.2 已加上下文白名单，党政语境（同级 / 国际先进 / 国际标杆 / 启示 / 经验 / 案例）±2 行内豁免

## 适用场景

- 政府工作报告
- 省 / 市 / 县发展规划
- 行业协会五年规划
- 咨询报告对照启示章节

## 关键提醒

「对标」在以下语境合法：

- ✓ 「推动人工智能模型对标国际先进水平」（政府工作报告原文）
- ✓ 「对标世界银行新营商环境评估框架」（政府工作报告原文）
- ✓ 「对标先进经验」「对标对表」（习近平讲话标准用法）
- ✓ 「对照启示」（咨询报告章节标题）

「对标」在以下语境是大厂黑话：

- ✗ 「我们对标头部产品 KPI」
- ✗ 「对标拼多多增速」
- ✗ 「内部赛马，对标行业大盘」

xuan-jiang v4.3 scan 通过 ±2 行扩窗自动识别——含「政府工作报告 / 党中央 / 二十大 / 同级 / 国际 / 启示 / 经验 / 案例」等关键词的"对标"豁免，含「KPI / 赛马 / 拉通 / 颗粒度 / 跑赢」等关键词的"对标"FAIL。
