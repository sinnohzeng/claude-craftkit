# IT 实物语境防火墙正向用例

机房按等保三级要求建设。网络架构遵循 GB/T 22239 国家标准，采用经典三层结构。

边界部署防火墙 2 台、入侵检测设备 2 台，覆盖南北向流量管控；内网核心交换机两台双机热备，对接业务服务器与数据库服务器。

Web 应用防火墙（WAF）部署在 DMZ 区，对外网请求做 SQL 注入、XSS、CC 等常见攻击的检测与拦截。NGFW 与 IPS 联动，对端口扫描和暴力破解告警。

针对协会内部 OA、邮件、CMS 等应用，统一通过堡垒机访问；防火墙规则按最小授权原则配置，每季度复盘一次访问控制列表。

期望：scan PASS（防火墙均落在 IT 实物语境，±2 行扩窗豁免）。
